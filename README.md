# Open ICS In Google Calendar

Open `.ics` calendar files in Google Calendar on macOS.

This project installs:

- a Folder Action for `Downloads`
- an app for opening `.ics` files directly

## Install

```sh
./install-open-ics-in-google-calendar.sh
```

Then attach `open - in Google Calendar.scpt` to `Downloads` in Folder Actions Setup.

![Folder Actions Setup](docs/folder-actions-setup.png)

## Use

- download an `.ics` file into `Downloads`, or
- double-click an `.ics` file

The file opens as a pre-filled Google Calendar event in your browser.
